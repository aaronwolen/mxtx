# mxtx: OpenMx Transcriber

## Installation

```r
# install.packages("devtools")
devtools::install_github("aaronwolen/mxtx")
```

## Example

```r
library(mxtx)
mxReport(list(AdeFit, AeFit, c(AdeFit, AeFit)))
```
